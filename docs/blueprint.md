# **App Name**: BandMate

## Core Features:

- Client Management: Register new clients with contact information and event history.
- Artist Management: Register new artists with profiles and performance details.
- Event Scheduling: Schedule new events, selecting clients and artists from the registry with date and time.
- Event Finance Tracking: Tracks finances to determine total income and other monetary goals.
- Dashboard Overview: Complete dashboard to overview incomes and outcomes.
- Reporting Module: Detailed reports, which includes movement and also graphics.
- Event Calendar: Visual event calendar with color-coded events for easy tracking.

## Style Guidelines:

- Background color: Light, desaturated cyan (#E0F7FA) to ensure that content is easily readable.
- Primary color: Moderate cyan (#4DD0E1) as it evokes both creativity and calm, fitting for both artists and organizers.
- Accent color: Deep blue (#1976D2) to provide visual interest and a clear visual hierarchy.
- Font: 'PT Sans', a versatile sans-serif for both headers and body text.
- Minimalist icons for menu items and interactive elements.
- Clean, intuitive dashboard layout for easy navigation.
- Smooth transitions between views and interactive elements.
